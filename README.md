# AirNuansa (LaporAir) - Management System

[![Framework](https://img.shields.io/badge/Laravel-11.x-red.svg)](https://laravel.com)
[![Stack](https://img.shields.io/badge/Stack-TALL-blue.svg)](https://tallstack.dev)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**AirNuansa** adalah platform manajemen distribusi air dan iuran warga berbasis web yang dirancang khusus untuk efisiensi administrasi di lingkungan perumahan (studi kasus: Nuansa Manisi Regency). 

Aplikasi ini mendigitalisasi proses pencatatan meteran air, pengelolaan tagihan bulanan, hingga pelaporan keuangan transparan bagi warga.

## 🚀 Key Features

* **Dashboard Analytics:** Ringkasan pendapatan bulanan dan status tunggakan warga secara real-time.
* **Customer Management:** Fitur *On/Off* status pelanggan untuk menangani rumah kosong/pindah agar tagihan tidak terakumulasi secara otomatis.
* **Flexible Billing:** Sistem pembagian tarif yang mendetail (Air, Sampah, dan Iuran Keamanan/IPL).
* **Financial Reporting:** Automasi laporan keuangan periode 2026 dengan kategori pendapatan yang terorganisir.
* **Resident Portal:** Tampilan tabel pembayaran yang memudahkan warga mengecek riwayat tagihan mereka.

## 🛠 Tech Stack

Aplikasi ini dibangun menggunakan **TALL Stack** untuk memberikan pengalaman *Single Page Application* (SPA) yang responsif namun tetap ringan:

* **Tailwind CSS** – Styling UI yang modern dan responsif.
* **Alpine.js** – Interaktivitas ringan di sisi frontend.
* **Laravel** – Backend framework yang robust dan aman.
* **Livewire** – Komponen frontend yang reaktif tanpa meninggalkan ekosistem PHP.
